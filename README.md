# Cooking Masterclass Catalogue

## Project Overview

Cooking Masterclass Catalogue is a responsive Vue.js single-page application that showcases cooking workshops hosted by professional chefs.

Users can:

* Browse available cooking classes
* View chef names, prices, and skill levels
* Save courses to a wishlist
* See sold out courses

This project was built using Vue 3 and Vite.

---

## Technologies Used

* Vue 3
* Vite
* JavaScript
* CSS

---

## Installation & Run Instructions

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/cooking-masterclass-catalogue.git
```

### 2. Open the project folder

```bash
cd cooking-masterclass-catalogue
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

### 5. Open in browser

Open the localhost link shown in the terminal.
